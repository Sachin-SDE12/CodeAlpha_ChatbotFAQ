# CodeAlpha FAQ Chatbot 🤖

A machine learning based FAQ chatbot built for CodeAlpha AI Internship Task 2.

## 🚀 Features
- Semantic search using `sentence-transformers`
- Pre-trained model: `all-MiniLM-L6-v2` 
- Cosine similarity for best FAQ matching
- Gradio web interface
- Handles out-of-scope questions

## 🛠️ Tech Stack
- Python 3.14
- Gradio 5.x
- Sentence-Transformers
- PyTorch
